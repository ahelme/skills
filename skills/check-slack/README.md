README for `check-slack` or 	`cs`  SKILL.md

# Requirements

- Slack channel with Claude Code plugin installed
- ~/bin/team-hear plugin - or another plugin to allow agent to post to Slack using a team-name set on a per dir basis (e.g. based on pantalk)
- .claude/teams-chat.local.md - setting team's emoji and name

# Configuration

- set name of plugin (replace `~/bin/team-hear` )
- set team identity in  `.claude/teams-chat.local.md`
